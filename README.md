# はんぐらびぃ主人公　ぐらびぃサンプル

- 動作環境 Unity2019.4.11f1

## 使い方
- `Scenes`フォルダー内の`SampleScene`を起動
- 開始して、`Animator`ウィンドウでパラメーターを設定してアニメを確認
